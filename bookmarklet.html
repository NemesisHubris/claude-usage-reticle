Install Options
Chrome Extension
Recommended
Auto-runs on Claude Usage with no script manager required. Settings live in the extension popup, not inside Claude.
  1. Download the project ZIP from GitHub.
  2. Unzip it and keep the folder somewhere stable.
  3. Open chrome://extensions, enable Developer mode, then click Load unpacked.
  4. Select the unzipped extension folder.
  5. Open claude.ai/settings/usage.
Download ZIP View Extension
Chrome/Edge/Brave: use chrome://extensions -> Developer mode -> Load unpacked.
Firefox testing: use about:debugging#/runtime/this-firefox -> Load Temporary Add-on -> select extension/manifest.json.
Updating the extension
  1. Download the latest ZIP.
  2. Replace your local project folder.
  3. Go back to chrome://extensions and click the reload button on Claude Usage Reticle.

What You See

How to read it

  • Blue marker marks your actual usage position.
  • Green label means you are under the expected pace.
  • Red label means you are over the expected pace.
  • Color intensity scales with how far off you are.

Notes

  • Updates on page changes, focus, and visibility changes.
  • Works with the session bar and weekly model bars.
  • Skips fixed-count bars such as Daily included routine runs.
  • Bookmarklet uses a small one-click injector with no settings panel.
  • The extension popup controls custom budget-window settings.
